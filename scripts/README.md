# Scripts Directory

This folder contains setup and utility scripts for the Supportta Bill Book application.

## Available Scripts

- `create-superadmin-direct.sql` - SQL script to create superadmin user
- `create-superadmin.ts` - TypeScript script to create superadmin (run with `npm run create-superadmin`)
- `rls-policies.sql` - Basic RLS policies (superseded by migration 003)
